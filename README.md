# WikiPlantas
WikiPlantas é um projeto criado a partir da minha paixão por plantas e do hábito de pesquisar frequentemente sobre seus cuidados, características e curiosidades. A ideia foi reunir essas informações de forma rápida, visual e intuitiva, criando uma espécie de “enciclopédia verde” pessoal, mas que pode ajudar qualquer pessoa interessada em plantas!
